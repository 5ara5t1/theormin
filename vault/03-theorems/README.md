# Theorems reproved from memory

Weekly verification: pick a theorem, reproduce its proof on a blank page, log here.

## Convention

- One note per theorem.
- Filename: descriptive kebab-case (`pythagoras.md`, `addition-commutativity-on-N.md`, `noether.md`).
- Log each attempt under `## Attempts` with date and outcome.

## Frontmatter

```yaml
---
type: theorem
theorem: "Addition is commutative on N"
book: tao-analysis-1
date: 2026-06-01          # most recent attempt
result: passed            # passed | partial | failed
time_minutes: 12
tags: [phase-1, analysis]
---
```

For multiple attempts, update `date`, `result`, and `time_minutes` to the latest; keep history in the body.

See [Operational Principle 4](../00-meta/curriculum.md) in the curriculum.
